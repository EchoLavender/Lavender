Markdown 简单语法

作者 : Lavender 时间 : 2017.11.09.

是否还在为html排版或者各种标签混杂不清等而感到烦乱，各种标签混杂不清。那就使用Markdown吧
”人生有很多坑，完全可以单脚一跃而过；可是大多数人，却偏偏要绕个大圈子。
对未知事物的恐惧，几乎是本能的反应吧？”
《Markdown 操作用户反馈》，沐沐周

首先要讲的是‘标题‘，也就是在标记语言HTML里面的<’h’>标签

这里为什么要用到 ‘’ 把h包括起来呢？

在「Markdown」里面，可以选择类似Atx形式的写法来进行，在行首插入1到6个 ‘ # ’,对应的标题则是1到6阶，例如：

# 这是 H1
## 这是 H2
###### 这是H6

如果你觉得有点不太习惯，你也可以选择类似<’div’>..<’/div’>这种闭合类Atx的形式，这纯粹这是美观用的，若觉得这样看上去舒适，你也可以在行尾加上’#’（行首的井字符的个数决定标题的阶数），例如：

# 这是 H1 #
## 这是 H2 ##
### 这是 H3 ######

其次到‘引用’、‘分割线’，对应的标签是<’br’>标签

至于引用的标签，说实话，我也想知道…..

在「Markdown」 标记区块应用中是使用类似 email 中用 ‘ > ‘ 的引用方式，如果你还熟悉在 email 信件中引言部分，你就知道怎么在「Markdown」 文件中建立一个区块引用，那会看起来像是你自己先断好行，然后在每行面前加上 ‘ > ‘ :

> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
> consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
> Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.
> Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
id sem consectetuer libero luctus adipiscing.
(特殊字符，比如 > 可以在前加入反斜杆‘\‘ 来插入普通的符号）

然后你会发现它实现的效果就是上述所示，如果你想偷懒的话，「Markdown」 也允许你偷懒只在整个段落的第一行最前面加上 > ：

> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.
Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
id sem consectetuer libero luctus adipiscing.

兼容 HTML

如果你觉得Markdown 能处理的有限，你可以加入一些 HTML 区块元素——比如 <div>、<table>、<pre>、<p>等标签，必须在它们的开始标签与结尾标签不能用制表符或空格来缩进。Markdown的生成器有足够智能，不会在 HTML 区块标签外加上不必要的<p>标签。

  • 例子如下，在 Markdown 文件里加上一段 HTML 表格 : 
$ 这是一个普通段落。
$
$ <table>
$    <tr>
$        <td>Foo</td>
$    </tr>
$ </table>

请注意，在 HTMl 区块标签间的 Markdown 格式语法将不会被处理。比如，你在 HTML 区块内使用 Markdown 样式的 *强调* 会没有效果。

HTML 的区段（行内）标签如<span>、<cite>、<del>、可以在 Markdown 的段落、列表或是标题里随意使用。依照个人习惯，甚至可以不用 Markdown 格式，而直接采用 HTML 标签来格式化.

代码区块

和程序相关的写作或是标签语言原始码通常会有已经排版好的代码区块，通常这些区块我们并不希望它以一般段落文件的方式去排版，而是按原来的样子显示， Markdown 会用 <pre>、和<code> 标签来把代码块包起来。

要在 Markdown 中建立代码区块很简单， 只要简单地缩进 4 个空格或是 1 个制表符就可以， 

$ 这是一个普通段落:
$ 这是一个代码区块
